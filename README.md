# Portfolio

2020 - present build of my portfolio site. Features a mix of paid work and personal projects. 

Presents work in one of three formats:

1. UX case study write-ups and reflection
2. UI & Front-end Project description
3. Live demo

This project replaces two previous iterations:

- 2014: a redesigned version of the college portfolio
- 2012: a college portfolio featuring web design and multimedia projects created in courses while pursuing a degreee in Graphic Design & Interactive Media

## Setup

Make sure `gulp` CLI is installed

```
npm install --global gulp-cli
```

Then, in a terminal:

- Dev mode: `gulp`
- Build mode: `gulp build`
- Run build locally: `gulp start`
